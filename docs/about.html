About

Ray Tomlinson was the first person to send an email between two computers using his how file-transfer protocol, CYPNET. Tomlinson is the person behind the now ubiquiots "@" symbol used to seperate a user's name fro mthe host name's computer. There was however one drawback in Tomlinson's initial breakthroughs, single messages could not be accessed without viewing all other messages a user received. Larry Roberts at ARPA later solved this problem with a program that allowed one to file messages, delete them and display a menu of messages. He called it an RD, short for "read". Eventually this program caught on and people were writing their own versions of it but there was no set standard. To make things more convenient messaging protocols were debated and developed and these grew into the simple mail transfer protocol or SMTP.
